Every input decomposes to the same neutral SPI, so any cell pairs with any output — Go in, Java out is one command. Output backends: Rust via the legacy hand-written module (its emit profile is held byte-identical by differential test); Java exists only as data — one JSON emit profile, no backend module, javac-checked in CI.
